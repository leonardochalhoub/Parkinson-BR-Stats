PROJECT OBJECTIVE:
The CNES (Cadastro Nacional de Estabelecimentos de Saúde) equipment registry 
provides comprehensive data on healthcare equipment infrastructure across Brazil.
This directly supports your manuscript's central thesis about neuroimaging access
disparities affecting Parkinson's disease diagnosis in the Brazilian public health
system (SUS).

================================================================================
  EQUIPMENT TYPES RELEVANT TO PARKINSON'S DIAGNOSIS
================================================================================

CRITICAL FOR DP DIAGNOSIS (Neuroimaging):
  ├─ MRI (CODEQUIP=42)  - Ressonância Magnética
  │  └─ Gold standard for detecting substantia nigra changes in DP
  │  └─ Can distinguish Parkinson's from atypical parkinsonian syndromes
  │
  └─ CT (CODEQUIP=38)   - Tomógrafo Computadorizado
     └─ Rules out structural mimics (tumors, strokes)
     └─ Shows atrophy patterns associated with DP

IMPORTANT FOR DP ASSESSMENT (Functional/Neurological):
  ├─ EEG (CODEQUIP=08)  - Eletroencefalografia
  │  └─ Functional brain activity assessment
  │
  ├─ Ultrasound (CODEQUIP=19) - Brain ultrasound increasingly used for screening
  │
  └─ Rehabilitation Equipment (TIPEQUIP=08)
     └─ PT/OT equipment for managing DP motor symptoms

SUPPORTING FOR DP PATIENTS:
  ├─ X-Ray (CODEQUIP=21)
  ├─ ECG (CODEQUIP=06) - Cardiac assessment (DP patients often have autonomic issues)
  └─ Lab Equipment (TIPEQUIP=01) - Baseline health assessment

================================================================================
  EQUIPMENT TYPE CATEGORIES IN CNES REGISTRY
================================================================================

TIPEQUIP CODES:
  01/1 - Diagnostic Laboratory (18 codes)
         Clinical pathology, hematology, biochemistry equipment
         Relevance: Baseline health assessment for DP patients

  02/2 - Diagnostic Imaging (15 codes) ★ CRITICAL
         X-Ray, CT, MRI, Ultrasound, Nuclear Medicine
         Relevance: PRIMARY imaging modality for DP diagnosis

  03/3 - Diagnostic Functional (18 codes) ★ IMPORTANT
         EEG, EMG, Spirometry, Audiometry
         Relevance: Neurological function assessment in DP

  04/4 - Therapeutic Hospital (2 codes)
         Operating rooms, ICU equipment, ventilators
         Relevance: Acute DP management and complications

  05/5 - Support Utilities (15 codes)
         Generators, sterilizers, water systems
         Relevance: Infrastructure enabling imaging services

  06/6 - Support Administrative (11 codes)
         Information systems, medical records
         Relevance: Data management of imaging facilities

  07/7 - Support Facilities (7 codes)
         Hospital beds, furniture, transport equipment
         Relevance: Facility operations

  08/8 - Therapeutic Rehabilitation (13 codes) ★ IMPORTANT
         Physical therapy and occupational therapy equipment
         Relevance: DP mobility and function management

  09/9 - Special Mobile (12 codes)
         Mobile clinics, portable equipment
         Relevance: Extended access to underserved areas

  10/10 - Special Preventive (4 codes)
          Community health screening equipment
          Relevance: Population-level DP screening

================================================================================
  KEY COLUMNS IN THE DATA
================================================================================

GEOGRAPHIC LOCATION:
  estado         - State code (UF) - e.g., "DF", "SP", "AC"
  CODUFMUN       - Municipality IBGE code
  REGSAUDE       - Health region
  DISTRSAN       - Sanitary district

EQUIPMENT IDENTIFICATION:
  TIPEQUIP       - Equipment type category (01-10)
  CODEQUIP       - Specific equipment code (e.g., 42=MRI, 38=CT)
  CNES           - Facility identifier
  TP_UNID        - Unit type (hospital, clinic, health center)

QUANTITY & STATUS:
  QT_EXIST       - Total quantity of equipment owned
  QT_USO         - Quantity currently in use/operational
  Difference reveals: Infrastructure capacity constraints

PUBLIC vs PRIVATE:
  IND_SUS        - Available for SUS public system (0=No, 1=Yes) ★
  IND_NSUS       - Available for private patients (0=No, 1=Yes) ★
  These directly validate your manuscript's claim: "28% in SUS, 72% private"

TIME PERIOD:
  ano            - Year (2020-2025)
  mes            - Month (01-12)
  COMPETEN       - Competency date (YYYYMM)
  Allows: Historical trend analysis of equipment availability

FACILITY TYPE:
  TPGESTAO       - Management type (E=State, M=Municipal, D=Direct)
  ESFERA_A       - Administrative sphere (F=Federal, E=State, M=Municipal, P=Private)
  PF_PJ          - Individual or organization managed

================================================================================
  ANALYSIS READY QUERIES
================================================================================

QUERY 1: Count MRI machines by state
   SELECT estado, COUNT(*) FROM cnes_eq 
   WHERE CODEQUIP = '42' 
   GROUP BY estado 
   ORDER BY COUNT(*) DESC

QUERY 2: MRI availability in SUS vs Private
   SELECT 
     estado,
     SUM(CASE WHEN IND_SUS=1 THEN 1 ELSE 0 END) as SUS_Available,
     SUM(CASE WHEN IND_NSUS=1 AND IND_SUS=0 THEN 1 ELSE 0 END) as Private_Only
   FROM cnes_eq 
   WHERE CODEQUIP = '42'
   GROUP BY estado

QUERY 3: Equipment operational status
   SELECT 
     CODEQUIP,
     SUM(QT_EXIST) as Total_Equipment,
     SUM(QT_USO) as In_Use,
     ROUND(100.0 * SUM(QT_USO) / SUM(QT_EXIST), 1) as Operational_Percent
   FROM cnes_eq
   WHERE CODEQUIP IN ('42', '38', '21', '19')  -- Neuroimaging devices
   GROUP BY CODEQUIP

QUERY 4: Regional disparity analysis (Southeast vs North)
   SELECT 
     estado,
     TIPEQUIP,
     COUNT(*) as Equipment_Count
   FROM cnes_eq
   WHERE TIPEQUIP IN ('02', '2')  -- Diagnostic Imaging category
   GROUP BY estado, TIPEQUIP
   ORDER BY estado

QUERY 5: Time series trend (equipment additions 2020-2025)
   SELECT 
     ano,
     mes,
     CODEQUIP,
     COUNT(*) as Equipment_Count
   FROM cnes_eq
   WHERE CODEQUIP IN ('42', '38', '21', '19')  -- Neuroimaging
   GROUP BY ano, mes, CODEQUIP
   ORDER BY ano, mes
